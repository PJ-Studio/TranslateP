## 需求描述
我现在准备增加一个单词本功能，目的在于保存每次翻译的内容到文件中，方便用户复习。

## 功能实现

### 菜单
1. 在 menu bar 窗口的“功能设置”分类里，剪贴板截图翻译下增加“自动保存到单词本”开关、“单词本路径设置”按钮和“单词本”按钮三个选项，注意 UI 风格和现有逻辑保持一致。
2. 当“自动保存到单词本”开关开启后，每次翻译都保存到同一个 md 文件中（没有就新建），以目标语言为文件分类索引，比如英译中，就新建或复用一个英文翻译文件，所有翻译内容都放入其中。
3. “单词本路径设置”选项右边有一个 sf symbol icon 按钮，点击后唤起文件选择器，让用户指定翻译结果文件存储路径。
4. 每次“自动保存到单词本”开关开启后，都要自动检查单词本路径是否设置，如果没有设置就自动唤起文件选择器让用户设置，做的事情与第三点一致，只是多了一个自动化。
5. 点击“单词本”选项按钮后，会弹出单词本窗口。

### 单词本
1. 翻译结果 md 中，写入源语言文本和目标语言文本单词，包括音标。提供朗读翻译前的单词朗读功能，复用现有的朗读逻辑。
2. 点击菜单窗口“单词本”选项按钮时，弹出单词本窗口。整体 UI 分为左右两边。左边以目标语言为分类，比如英译中和中译英就以英文和中文列出两个选项。用户选择“英文”选项，右边部分就出现所有查询过的英文单词，并把第一点实现的功能放入其中。
3. 单词本以查询日期为分类聚合，每一天查询的内容都聚合在一个 section 下。
4. 单词本提供删除某个单词功能。

## 其他要求

2. UI 要按照目前的 app 整体风格统一。
